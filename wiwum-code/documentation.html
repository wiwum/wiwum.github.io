basic syntax:

each line is split into three sections: part 1, part 2, and part 3. Part 1 selects what type of thing is going to be changed, such as the style. Part 2 specifies what specific thing is going to be changed. Part 3 specifies how something will be changed. Specifics can be found below. you can go back to the main page here.

main page



display ------ changes the content of the output

text ------ changes the text in the output

example ---> part 1: display part 2: text part 3: this text is displayed

NOTE: part 3 can be any text you want

image ------ changes the image that is desplayed

example ---> part 1: display part 2: image part 3: ./example.jpg

NOTE: part 3 can be any image url you want jus copy and past it (don't put "./" in front of urls from the web)

moveOutput ------ moves the output to the right

example ---> part 1: display part 2: moveOutput part 3: 100px

NOTE: part three can be any number and the unit is pixels (px)

addStyle ------ changes the style(colors) of the of the output

backgroundColor ------ changes the background color of the output

example ---> part 1: addStyle part 2: backgroundColor part 3: red

NOTE: part three can be any color name valid in css or a hexidecimal value (for hexidecimal values put a # infront)

textColor ------ changes the text color of the output

example ---> part 1: addStyle part 2: textColor part 3: red

NOTE: part three can be any color name valid in css or a hexidecimal value (for hexidecimal values put a # infront)

textSize ------ changes the size of the text

example ---> part 1: addStyle part 2: textSize part 3: 20px

NOTE: part three can be any number and the unit is pixels (px)

changeText ------ changes the text after a certain amount of time

example ---> part 1: changeText part 2: 3000 part 3: this text is displayed after three seconds

NOTE: part 2 is the number of miliseconds after the program is run that the text will be displayed

changeImage ----- changes the image after a certain amount of time

example ---> part 1: changeImage part 2: 3000 part 3: ./newImage.jpg

NOTE: part 2 is the number of miliseconds after the program is run that the text will be displayed
